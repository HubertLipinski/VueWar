# VueWar

#### Live demo: https://vue-war.now.sh/

Simple game made with <br/>
<ul>
    <li>Vue.js</li>
    <ul>
        <li>Vuex</li>
        <li>Vue-router</li>
        <li>Buefy</li>
    </ul>
    <li><a href="https://deckofcardsapi.com/">Cards API</a></li>
</ul>

#### Project setup
```
git clone https://github.com/HubertLipinski/VueWar.git
cd VueWar
npm install
npm run serve
```
